Modelo de la caja, propiedades, box sizing y Flexbox

Modelo de la caja

El modelo de la caja es un concepto fundamental en CSS que describe cómo se representan los elementos en una página web. Cada elemento se considera una caja rectangular que tiene un contenido, un relleno (padding), un borde (border) y un margen (margin).

Propiedades del modelo de la caja

  • width: Define el ancho del contenido.
  • height: Define la altura del contenido.
  • padding: Espacio entre el contenido y el borde.
  • border: El borde alrededor del padding.
  • margin: Espacio fuera del borde.

Box Sizing

La propiedad box-sizing permite cambiar el comportamiento del modelo de la caja. Por defecto, el ancho y alto de un elemento solo incluyen el contenido. Con box-sizing: border-box, el ancho y alto incluyen el padding y el borde.

Flexbox

Flexbox es un modelo de diseño que permite distribuir espacio entre los elementos de una interfaz y alinear elementos dentro de un contenedor. Es especialmente útil para crear diseños responsivos.
